# Discord-Music-Bot
> A simple Discord music bot.

## Setup
1. Download and install python version `3.6` or higher
2. Install discord.py

Linux/macOS
```sh
$ python3 -m pip install -U "discord.py[voice]"
```
Windows
```sh
$ py -3 -m pip install -U discord.py[voice]
```

3. Install wavelink

```sh
$ pip install wavelink
```
